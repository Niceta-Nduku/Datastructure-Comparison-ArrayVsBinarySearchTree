# Datastructure-Comparison-ArrayVsBinarySearchTree

Class assignent to test the performance of a traditional array against a binary search tree

To run PowerArrayApp or PowerBSTApp in terminal,
make in the directory that contains the make file, 
open the bin directory and run the app. For example

	java PowerArrayApp

which will print all the date/times

Alternatively, to run all the test cases,
run the bash script in the practical directory

	/bin/bash TestCases.sh

read the text files that are generated and 
	
	rm -f *.txt

To run the experiments 

	/bin/bash Experiment1.sh
	/bin/bash Experiment2.sh
	
The data will be generated to .xlsx files
